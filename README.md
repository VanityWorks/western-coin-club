# Western Coin Club – Frontend Demo

A visual demo website for the Western Coin Club, built with Vite + React (SPA).

## Theme

- **Colours:** White and black with South African flag accent pops (green, gold, red, blue)
- **Fonts:** Cormorant Garamond (headings), Source Sans 3 (body)

## Run

```bash
npm install
npm run dev
```

Open http://localhost:5173 (or the port shown in the terminal).

## Build

```bash
npm run build
```

## Pages

- **Home** – Hero, latest news, benefits, audience, collecting categories, testimonials, pricing, CTA
- **About** – Mission, what we do, history
- **Membership** – Signup form (demo only, no submission)
- **News** – Article list with search and category filters
- **Consulting** – Services overview and request form
- **Forum** – Category list (placeholder)
- **Contact** – Contact form and details

All forms are frontend-only and do not submit data.
